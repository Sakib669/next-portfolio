# Shafiqul Islam Sakib - Portfolio Website

A professional, animated portfolio website showcasing my work as a Full Stack MERN Developer. Built with Next.js 15, TypeScript, and Tailwind CSS with custom animations and interactive effects.

## ✨ Key Features

### 🎬 Professional Animations
- **Loading Screen**: Animated gradient with progress bar
- **Typing Animation**: Rotating job titles with cursor effect
- **Particle System**: Interactive background with connecting particles
- **Counter Animations**: Numbers count up on scroll
- **Skill Bars**: Animated progress bars with shimmer effects
- **Scroll Animations**: Elements fade/slide in as you scroll
- **Hover Effects**: Lift, scale, and glow effects throughout
- **Custom Cursor**: Animated cursor trail (desktop only)
- **Page Transitions**: Smooth transitions between pages
- **Gradient Animations**: Shifting color gradients on text and backgrounds

### 🎨 Design Features
- Modern, clean, and professional design
- Dark mode support with smooth transitions
- Fully responsive (mobile, tablet, desktop)
- Glassmorphism effects
- Gradient backgrounds and text
- Custom scrollbar styling
- Smooth scroll behavior
- Professional color scheme (Blue to Purple gradients)

### 📄 Pages

- Animated hero section with typing effect

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

### Production Build

```bash
npm run build
npm start
```

## 🎨 Customization

3. **Animations**: Adjust timing in `globals.css`
4. **Social Links**: Update URLs in components
5. **Projects**: Add more projects in About page
6. **Skills**: Modify skill levels and categories
