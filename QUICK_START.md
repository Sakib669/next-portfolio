## 🔧 Quick Customization

### Update Your Links

1. **GitHub**: Replace `https://github.com` with your actual GitHub URL
2. **LinkedIn**: Replace `https://linkedin.com` with your LinkedIn profile
3. **Portfolio**: Add your portfolio website URL

### Add Your Photo

Replace the "SS" placeholder in `app/about/page.tsx`:
```tsx
// Current: <div className="text-white text-6xl font-bold">SS</div>
// Replace with: <Image src="/your-photo.jpg" alt="Profile" ... />
```

### Update Project Links

Add live links and GitHub repos to your projects in:
- `app/page.tsx` (Featured projects)
- `app/about/page.tsx` (Detailed projects)
